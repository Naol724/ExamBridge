# ExamBridge HU

A centralized academic knowledge and collaboration platform for CCI students at Haramaya University.

## Tech Stack

- **Backend**: Node.js with Express
- **Database**: MongoDB (Atlas)
- **File Storage**: Firebase Storage
- **Real-time**: Socket.IO
- **Frontend**: React.js with Tailwind CSS
- **Authentication**: JWT-based login/register system

## Features

- Year-based and semester-based academic content organization
- Real-time chat system per course
- PDF note storage and sharing
- Exit exam preparation materials
- Admin content management system
- Student and admin authentication

## Getting Started

1. Clone the repository
2. Set up backend dependencies
3. Configure environment variables
4. Set up frontend dependencies
5. Run both backend and frontend servers
